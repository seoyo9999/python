# CH1 파이썬 문법


## 자료형
#### 숫자형
* 정수 : 123, -20, 0
* 실수 : 123.45, -4321.5, 6.08e9
* 8진수 : 0o456, 0o123
* 16진수 : 0xFF, 0x0D, 0x0A


#### 변수  
* 문자 또는 밑줄로 시작(beta, _kim)
* 대소문자를 구분한다.(sum, Sum, SUM)
* 영문자, 숫자(A-z, 0-9)
* 파이썬 키워드는 사용 불가


#### 문자열
1. 큰 따옴표 : "Hello World! it\'s"
2. 작은 따옴표 : '대한민국'
3. 큰 따옴표 3 : """Hello!"""
4. 작은 따옴표 3 :
'''Life is too short. You need python'''


#### 데이터 타입
+ 텍스트
+ 숫자
+ 불(bool)
